소개 : 패키지 관광중 가이드와 관광객의 커뮤니케이션을 원활히 할 수 있도록 도와주는 앱

제작기간 : 2018. 07 ~

역할 : 안드로이드 앱 파트를 맡았으며, 디자인 1명, 중국어 번역 및 기획 2명 총 4명이 팀을 이루었습니다.

기능

  • 핫스팟을 이용한 UDP소켓 통신으로 가이드앱에서 목소리를 관광객 앱쪽으로 녹음과 동시에 전달
  • 가이드앱에서 작성한 스케줄이 변경시 관광객앱쪽으로 데이터를 보내어 업데이트
  • 가이드와 관광객들의 전체 단톡방을 구현하여 채팅가능
  • SharedPreference를 이용하여 로그인정보 기억하여 로그인 유지
  • Sqlite를 이용하여 메시지, 스케줄 데이터 저장
  • IntentService를 이용한 데이터 receive작업과 LocalBroadCastReceiver를 이용하여 데이터 전송
  • poi라이브러리를 이용하여 스케줄이 작성된 엑셀파일 읽어올 수 있음

사용라이브러리 : StickyHeaderRecyclerView

배운점 : Kotlin을 이용하여 제작한 앱입니다, map, filter등의 스트림함수와 apply, run등의 범위지정 함수, ? ,!! 을 이용한 널체크, by lazy를 이용한 지연 초기화 등을 사용하는 방법을 배웠습니다.

bitbucket

소개 : 제주도 관광상품을 직접 만들고 즐겨찾기한 상품내부의 스케쥴을 바꿀 수 있는 앱

기업체에 제작한 앱을 인수받아 유지보수를 하였습니다.

유지보수기간 : 2017. 07 ~ 2018. 06

역할 : 앱과 서버의 유지보수를 맡았으며 디자인 1명, 개발보조 1명, 기획 및 서버데이터관리 4명으로 총 7명으로 팀을 이루었습니다.

기능

  • 서버에 있는 제주도의 모든 관광지 및 음식점, 숙박시설 데이터를 이용하여 관광상품 제작가능.
  • 만들어져 있는 상품을 가져와 원하는 스케쥴을 삽입 및 수정가능.
  • 자신이 만든 상품을 다른 사용자에게 공유 및 리뷰가능.
  • 제주도의 모든 관광지의 대한 정보를 RecyclerView로 제공
  • 제주도의 음식점, 숙박시설의 대한 정보를 RecyclerView로 제공
  • 관광상품 및 관광지, 음식점, 숙박시설의 위치를 지도에 표시
  • VR sdk를 이용하여 드론을 이용하여 찍은 관광지 동영상의 모습을 보여줌(예정)

사용라이브러리 및 기술: Kakao Login, Google Map, google VR, DraagableRecyclerView, Glide, Tedpermission, Mysql, tomcat, mybatis

배운점 : AsyncTask를 이용한 서버와의 데이터통신 이해, CustomView의 이해, Viewholder패턴의 이해, Callback의 이해, 유지보수를 통한 코드분석능력 향상

성과 : 한국산업기술진흥원에서 주관하는 2017 공학페스티벌 참가하여 한국여성공학기술인협회장상을 수상하였습니다.

소개 : 자취생을 위한 빨래, 청소, 냉장고 관리를 위한 notification 알림 앱

제작기간 : 2017. 06 ~ 2017. 08

역할 : 앱의 모든부분을 관리하였고 , 디자인 1명, 개발보조 2명로 총 4명으로 팀을 이루었습니다.

기능

  • navigation을 이용하여 빨래, 청소, 냉장고의 스케쥴을 관리하는 Fragment에 접근
  • 설정하는 각 스케쥴마다 경고기간을 넣어 '보통,경고,시급'으로 등급을 나누어 각각의 페이저에 리스트로 표시를 해줌
  • notification에 경고, 시급 등급의 스케쥴 갯수를 표시하고 클릭 시 앱이 켜짐
  • 리스트의 한 아이탬을 swipe할 시 재갱신버튼과 삭제버튼이 나타나고 재갱신을 할 시 기존의 정보를 통해 새로운 일자가 들어감
  • broadcastReceiver를 이용하여 화면이 켜질 때 날자변경 감지 및 notification 갱신

사용라이브러리 : SwipeableRecyclerView

배운점 : Fragment의 사용방법 습득, AdapterView의 사용방법 습득, Sqlite의 사용방법 습득
